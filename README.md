# Spin Game

A single-page slot-machine-style randomiser for GitHub Pages.

## Upload to GitHub Pages

1. Create a new GitHub repository, for example `spin-game`.
2. Upload these files to the repository root:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `icon.svg`
3. In GitHub, go to **Settings → Pages**.
4. Under **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
5. Save.
6. Wait a minute or two, then open the Pages URL GitHub gives you.

On iPhone/iPad, open the site in Safari, tap **Share → Add to Home Screen**.
